# C++ 그래프 (Graph)

---

## 그래프란?

> **정점(Vertex)과 간선(Edge)으로 이루어진 자료구조**

```
정점(V) = 노드, 데이터를 담는 곳
간선(E) = 정점과 정점을 연결하는 선
```

> 💡 **비유:** 도시(정점)와 도로(간선) — 도시들이 도로로 연결된 지도

---

## 그래프 종류

|종류|설명|표현|
|---|---|---|
|**무방향 그래프**|간선에 방향 없음|A — B|
|**방향 그래프**|간선에 방향 있음|A → B|
|**가중치 그래프**|간선에 비용/거리 있음|A —5— B|
|**비가중치 그래프**|간선에 가중치 없음|A — B|

---

## 그래프 표현 방법 2가지

### 방법 1: 인접 행렬 (Adjacency Matrix)

> **2차원 배열**로 표현 — `graph[i][j] = 1` 이면 i→j 간선 존재

```cpp
int graph[5][5] = {0};   // 정점 5개, 초기화

// 무방향 그래프: 양방향 모두 표시
graph[1][2] = 1;
graph[2][1] = 1;

// 가중치 그래프: 1 대신 가중치 값
graph[1][2] = 5;
graph[2][1] = 5;
```

```
     1  2  3  4
  1 [0, 1, 0, 1]
  2 [1, 0, 1, 0]
  3 [0, 1, 0, 1]
  4 [1, 0, 1, 0]
```

**장단점:**

```
✅ 두 정점의 연결 여부 O(1) 확인
✅ 구현 단순
❌ 메모리 O(V²) → 정점 많으면 비효율
❌ 간선 적어도 전체 배열 탐색
```

---

### 방법 2: 인접 리스트 (Adjacency List)

> **벡터 배열**로 표현 — 각 정점마다 연결된 정점 목록 유지

```cpp
#include <vector>
using namespace std;

vector<int> graph[5];   // 정점 5개

// 무방향 그래프
graph[1].push_back(2);
graph[2].push_back(1);

graph[1].push_back(4);
graph[4].push_back(1);

// graph[1] = {2, 4}
// graph[2] = {1}
// graph[4] = {1}
```

**가중치 있는 경우 — pair 사용:**

```cpp
vector<pair<int,int>> graph[5];  // {연결정점, 가중치}

graph[1].push_back({2, 5});   // 1→2, 비용 5
graph[2].push_back({1, 5});   // 2→1, 비용 5
```

**장단점:**

```
✅ 메모리 O(V + E) → 간선 적을 때 효율적
✅ 연결된 정점만 순회
❌ 두 정점 연결 여부 확인 O(V)
```

---

### 인접 행렬 vs 인접 리스트 비교

|구분|인접 행렬|인접 리스트|
|---|---|---|
|메모리|O(V²)|O(V + E)|
|연결 확인|**O(1)**|O(V)|
|간선 순회|O(V²)|**O(E)**|
|적합한 경우|간선 많을 때 (밀집)|간선 적을 때 (희소)|
|구현 난이도|쉬움|보통|

> 💡 **백준에서는 대부분 인접 리스트 사용** (메모리 효율)

## BFS (너비 우선 탐색)

> **가까운 정점부터 탐색** — Queue 사용

```
시작 정점 → 인접한 정점 전부 → 그 다음 인접한 정점 전부 → ...
```

```cpp
#include <queue>
#include <vector>
using namespace std;

vector<int> graph[1001];
bool visited[1001];

void bfs(int start) {
    queue<int> q;
    q.push(start);
    visited[start] = true;

    while (!q.empty()) {
        int cur = q.front();
        q.pop();
        cout << cur << " ";

        for (int next : graph[cur]) {
            if (!visited[next]) {
                visited[next] = true;
                q.push(next);
            }
        }
    }
}
```

**특징:**

```
✅ 최단 경로 보장 (비가중치 그래프)
✅ 레벨 단위 탐색
시간복잡도: O(V + E)
```

## DFS (깊이 우선 탐색)

> **한 방향으로 끝까지 탐색 후 되돌아옴** — Stack 또는 재귀 사용

```
시작 정점 → 인접 정점 하나 선택 → 끝까지 → 백트래킹
```

### 재귀로 구현

```cpp
vector<int> graph[1001];
bool visited[1001];

void dfs(int cur) {
    visited[cur] = true;
    cout << cur << " ";

    for (int next : graph[cur]) {
        if (!visited[next]) {
            dfs(next);
        }
    }
}
```

### 스택으로 구현

```cpp
void dfs(int start) {
    stack<int> s;
    s.push(start);

    while (!s.empty()) {
        int cur = s.top();
        s.pop();

        if (visited[cur]) continue;
        visited[cur] = true;
        cout << cur << " ";

        for (int next : graph[cur]) {
            if (!visited[next]) s.push(next);
        }
    }
}
```

**특징:**

```
✅ 메모리 효율 (재귀/스택)
✅ 경로 존재 여부 판단
❌ 최단 경로 보장 안 함
시간복잡도: O(V + E)
```

---

## BFS vs DFS 비교

|구분|BFS|DFS|
|---|---|---|
|자료구조|Queue|Stack / 재귀|
|탐색 순서|가까운 것 먼저|깊은 것 먼저|
|최단 경로|✅ 보장|❌ 보장 안 함|
|메모리|많이 사용|적게 사용|
|주요 용도|최단 거리, 레벨 탐색|경로 탐색, 사이클 감지|

## 전체 코드 구조 (백준 기준)

```cpp
#include <iostream>
#include <vector>
#include <queue>
using namespace std;

const int MAX = 1001;
vector<int> graph[MAX];
bool visited[MAX];
int n, m;   // 정점 수, 간선 수

void bfs(int start) { ... }
void dfs(int cur)   { ... }

int main() {
    cin >> n >> m;

    for (int i = 0; i < m; i++) {
        int a, b;
        cin >> a >> b;
        graph[a].push_back(b);
        graph[b].push_back(a);  // 무방향이면 양방향
    }

    // 시작
    bfs(1);
    // 또는
    dfs(1);
}
```

## 자주 쓰는 패턴

### 연결 요소 개수 세기

```cpp
int cnt = 0;
for (int i = 1; i <= n; i++) {
    if (!visited[i]) {
        dfs(i);   // 또는 bfs(i)
        cnt++;    // 새로운 연결 요소
    }
}
cout << cnt;
```

### 최단 거리 구하기 (BFS)

```cpp
int dist[MAX];
fill(dist, dist + MAX, -1);

void bfs(int start) {
    queue<int> q;
    q.push(start);
    dist[start] = 0;

    while (!q.empty()) {
        int cur = q.front(); q.pop();

        for (int next : graph[cur]) {
            if (dist[next] == -1) {
                dist[next] = dist[cur] + 1;
                q.push(next);
            }
        }
    }
}
```

### 사이클 감지 (DFS)

```cpp
bool hasCycle = false;

void dfs(int cur, int parent) {
    visited[cur] = true;

    for (int next : graph[cur]) {
        if (!visited[next]) {
            dfs(next, cur);
        } else if (next != parent) {
            hasCycle = true;  // 이미 방문 + 부모 아님 = 사이클
        }
    }
}
```

## 핵심 정리

```
그래프 표현:
  인접 행렬  → graph[i][j], 연결 확인 O(1), 메모리 O(V²)
  인접 리스트 → graph[i] = {}, 메모리 O(V+E), 백준에서 주로 사용

탐색:
  BFS → Queue, 최단 경로 보장, 레벨 단위
  DFS → Stack/재귀, 경로 탐색, 사이클 감지

공통 주의:
  visited[] 배열로 방문 체크 필수
  무방향이면 양방향 모두 push_back
  방향 그래프면 한 방향만 push_back
```
