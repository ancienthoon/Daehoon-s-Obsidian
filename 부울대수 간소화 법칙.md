## 기본 규칙 8가지

|규칙|AND|OR|
|---|---|---|
|0/1|A · 0 = 0|A + 1 = 1|
|0/1|A · 1 = A|A + 0 = A|
|멱등|A · A = A|A + A = A|
|보수|A · A' = 0|A + A' = 1|

> 💡 왼쪽 AND 규칙에서 연산자·값을 반대로 바꾸면 오른쪽 OR 규칙 → **쌍대성의 원리**

---

## 3가지 기본 법칙

### 교환 법칙

```
A + B = B + A
A · B = B · A
```

### 결합 법칙

```
(A + B) + C = A + (B + C)
(A · B) · C = A · (B · C)
```

### 분배 법칙

```
A(B + C) = AB + AC          ← 일반 수학과 동일
A + BC = (A+B)(A+C)         ← 부울 대수만의 특수 규칙!
```

## 추가 규칙 (간소화에 자주 쓰임)

```
A'' = A                     규칙 9  이중 보수
A + AB = A                  규칙 10 흡수 법칙
A + A'B = A + B             규칙 11 합의 법칙
(A+B)(A+C) = A + BC         규칙 12 분배 역이용
```

#### 증명 흐름만 기억하기

```
규칙 10: A + AB = A(1+B) = A·1 = A
규칙 11: A + A'B = (A+A')(A+B) = 1·(A+B) = A+B
규칙 12: (A+B)(A+C) → 전개 후 규칙 10 두 번 적용 → A+BC
```

## 드모르간의 정리 ⭐

```
(AB)' = A' + B'       AND → NOT → OR로 분리
(A+B)' = A' · B'      OR  → NOT → AND로 분리
```

**암기법:** 괄호 풀면서 안쪽 연산자 반대로(AND↔OR), 각 변수에 NOT 붙이기

```
확장:
(ABC...)' = A' + B' + C' + ...
(A+B+C+...)' = A' · B' · C' · ...
```

## 간소화 전략 요약

```
① 공통 변수 묶기     AB + AC = A(B+C)
② 보수 쌍 이용       A+A'=1, A·A'=0
③ 흡수 법칙          A + AB = A
④ 항 추가 후 묶기    A = A + A  (같은 항 두 번 써도 됨)
⑤ 드모르간           복잡한 NOT 처리
```
