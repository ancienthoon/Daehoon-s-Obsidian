# ⚖️ [C++] 정렬의 핵심, compare 함수 마스터

> [!info] **`compare` 함수의 정체** `sort(시작, 끝, 규칙)`에서 세 번째 인자로 들어가는 **'규칙'**입니다. "두 원소 `a`와 `b`가 있을 때, **누가 앞에 서야 할지**"를 `true`나 `false`로 알려주는 역할을 합니다.

---

## 1. 기본 원리: "앞에 서고 싶으면 true를 줘!"

`compare(a, b)` 함수에서 `true`를 반환한다는 건, **"a가 b보다 앞에 오는 게 맞아!"**라고 컴퓨터에게 말해주는 것과 같습니다.

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

// 오름차순 (작은 게 앞)
bool ascend(int a, int b) {
    return a < b; // a가 b보다 작으면 true -> a를 앞에 세움
}

// 내림차순 (큰 게 앞)
bool descend(int a, int b) {
    return a > b; // a가 b보다 크면 true -> a를 앞에 세움
}

int main() {
    ios_base::sync_with_stdio(false); cin.tie(NULL); cout.tie(NULL);

    vector<int> v = {3, 1, 4, 1, 5};
    sort(v.begin(), v.end(), descend); // 내림차순 정렬

    for(int x : v) cout << x << " "; // 5 4 3 1 1
    return 0;
}
```

## 2. [필수] 구조체 정렬 활용하기 (Baekjoon 스타일)

백준 골드 티어로 가기 위해서는 여러 조건이 섞인 **구조체 정렬**이 필수입니다. 예를 들어, **"성적은 높은 순으로, 성적이 같다면 번호가 빠른 순으로"** 정렬하는 상황입니다.

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

using namespace std;

struct student {
    string name;
    int score;
    int id;
};

// 정렬 규칙: 1순위 성적(내림차순), 2순위 번호(오름차순)
bool compare_students(const student& a, const student& b) {
    if (a.score != b.score) {
        return a.score > b.score; // 성적이 다르면 높은 사람이 앞
    }
    return a.id < b.id; // 성적이 같으면 번호가 작은 사람이 앞
}

int main() {
    ios_base::sync_with_stdio(false); cin.tie(NULL); cout.tie(NULL);

    vector<student> v = {
        {"kim", 90, 1},
        {"lee", 95, 2},
        {"park", 90, 3}
    };

    sort(v.begin(), v.end(), compare_students);

    for(auto s : v) {
        cout << s.name << " (" << s.score << ", " << s.id << ")\n";
    }
    /* 결과:
       lee (95, 2)
       kim (90, 1)
       park (90, 3)
    */

    return 0;
}
```

# 🔢 에라토스테네스의 체 (Sieve of Eratosthenes)

> [!info] **핵심 개념** "소수를 찾는 것이 아니라, **소수가 아닌 것들(배수)을 지워나가는 것**"입니다. 마치 체(Sieve)로 가루를 걸러내듯, 합성수들을 걸러내고 남은 것들이 바로 소수가 됩니다.

---

## 1. 작동 원리 (중학생도 이해하는 3단계)

1. 2부터 소수를 구하고자 하는 구간의 모든 수를 나열합니다.
    
2. 아직 지워지지 않은 수 중 가장 작은 수(소수)를 찾습니다.
    
3. **그 수의 배수들을 모두 지웁니다.** (자기 자신은 제외)
    
4. 이 과정을 반복하면 지워지지 않고 남은 수들이 소수가 됩니다.
    

---

## 2. C++ 최적화 코드

백준 문제를 풀 때 바로 사용할 수 있는 형태입니다. `bool` 배열을 사용하여 메모리를 아끼는 것이 '최고의 결과'입니다.

```cpp
#include <iostream>
#include <vector>

using namespace std;

// 최대 범위 설정 (예: 1,000,000)
bool is_prime[1000001];

void sieve(int n) 
{
    // 1. 모든 수를 소수로 초기화 (0과 1은 제외)
    for (int i = 2; i <= n; i++) is_prime[i] = true;

    // 2. n의 제곱근까지만 확인해도 충분합니다 (최적화)
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            // 3. i가 소수라면, i의 배수들을 모두 제거
            for (int j = i * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
}

int main() 
{
    ios_base::sync_with_stdio(false); cin.tie(NULL); cout.tie(NULL);

    int n = 100; // 100까지의 소수 구하기
    sieve(n);

    for (int i = 2; i <= n; i++) 
        if (is_prime[i]) cout << i << " ";

    return 0;
}
```

# 📦 C++ STL: pair (데이터 쌍)

> [!abstract] **핵심 정의**
> 
> - **데이터 쌍:** 서로 연관된 두 개의 데이터를 하나로 묶어주는 구조체 템플릿입니다.
>     
> - **서로 다른 타입:** 첫 번째 데이터와 두 번째 데이터의 자료형이 달라도 상관없습니다.
>     
> - **헤더 파일:** `<utility>` (단, `<vector>`나 `<algorithm>` 등을 불러오면 자동으로 포함되기도 합니다.)
>     

---

## 1. 선언 및 초기화

`pair`는 첫 번째 데이터를 `first`, 두 번째 데이터를 `second`라는 이름으로 저장합니다.

### ✅ 기본적인 방법

```cpp
pair<int, string> p;
p.first = 1;
p.second = "apple";

// 선언과 동시에 초기화
pair<int, int> coord = {10, 20}; 
```

### ✅ `make_pair` 함수 사용

데이터 타입을 직접 명시하지 않고도 쌍을 만들 수 있습니다.

```cpp
auto p = make_pair(3.14, "pi");
```

---

## 2. 왜 사용하는가? (비유와 특징)

- **비유:** 신발 상자에 '왼쪽 신발'과 '오른쪽 신발'을 담는 것과 같습니다. 서로 짝이 맞는 데이터를 잃어버리지 않게 하나로 묶어주는 역할을 합니다.
    
- **비교 (pair vs struct):**
    
    - 간단히 두 개만 묶을 때는 **`pair`**가 선언이 훨씬 빠르고 간편합니다.
        
    - 세 개 이상의 데이터를 묶거나, 각 멤버에 의미 있는 이름(예: `age`, `name`, `score`)을 붙이고 싶을 때는 **`struct`**가 유리합니다.
        

---

## 3. 실전 활용: 정렬 (Sorting)

`pair`의 가장 큰 장점 중 하나는 **기본적인 비교 연산자**가 정의되어 있다는 점입니다.

`vector<pair<int, int>>`를 정렬하면 다음과 같은 우선순위로 정렬됩니다.

1. **`first`** 기준 오름차순
    
2. 만약 `first`가 같다면 **`second`** 기준 오름차순
    

```cpp
#include <iostream>
#include <vector>
#include <utility>
#include <algorithm>
#include <string>

using namespace std;

int main() {
    ios_base::sync_with_stdio(false); cin.tie(NULL); cout.tie(NULL);

    // 좌표계 관리 예시
    vector<pair<int, int>> points;
    points.push_back({3, 4});
    points.push_back({1, 2});
    points.push_back({3, 1});

    // 정렬 시 실행 결과: (1, 2) -> (3, 1) -> (3, 4)
    sort(points.begin(), points.end());

    for (auto p : points) {
        cout << "(" << p.first << ", " << p.second << ")\n";
    }

    return 0;
}
```

## 🎓 편입 면접 대비: 핵심 질문

> [!question] **`pair`와 `tuple`의 차이점은 무엇인가요?**
> 
> **Answer:** "`pair`는 정확히 **두 개**의 데이터만 묶을 수 있지만, C++11부터 도입된 `tuple`은 **세 개 이상**의 데이터를 묶을 수 있습니다. `pair`는 `.first`, `.second`로 접근하고 `tuple`은 `get<index>(t)` 형식을 사용합니다."

> [!question] **`map` 컨테이너와 `pair`는 어떤 관계인가요?**
> 
> **Answer:** "`std::map`에 데이터를 저장할 때, 각 원소는 내부적으로 `pair<const Key, Value>` 형태로 저장됩니다. 그래서 `map`을 순회할 때 반복자(iterator)를 통해 `it->first`로 키에 접근하고 `it->second`로 값에 접근하게 됩니다."
