## 1. 이진 탐색 (Binary Search)

> **정렬된 배열에서 중간값과 비교하며 탐색 범위를 절반씩 줄이는 방법**

```
시간복잡도: O(log N)
전제 조건:  배열이 반드시 정렬되어 있어야 함
```

### 동작 원리

```
[1, 3, 5, 7, 9, 11, 13]  target = 7

① mid = (0+6)/2 = 3 → arr[3] = 7 → 찾음!

target = 9
① mid = 3 → arr[3] = 7 < 9 → 오른쪽 탐색
② mid = 5 → arr[5] = 11 > 9 → 왼쪽 탐색
③ mid = 4 → arr[4] = 9 → 찾음!
```

### 기본 구현

```cpp
#include <iostream>
#include <vector>
using namespace std;

int binarySearch(vector<int>& arr, int target) {
    int lo = 0, hi = arr.size() - 1;

    while (lo <= hi) {
        int mid = lo + (hi - lo) / 2;  // 오버플로우 방지

        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) lo = mid + 1;
        else hi = mid - 1;
    }
    return -1;  // 없음
}
```

> ⚠️ `mid = (lo + hi) / 2` 는 오버플로우 위험  
> 반드시 `mid = lo + (hi - lo) / 2` 사용

### STL 활용

```cpp
#include <algorithm>
vector<int> v = {1, 3, 5, 7, 9};

// 존재 여부
bool found = binary_search(v.begin(), v.end(), 7);  // true

// 첫 번째 ≥ target 위치
auto it = lower_bound(v.begin(), v.end(), 6);  // 7 가리킴
int idx = it - v.begin();  // 3

// 첫 번째 > target 위치
auto it2 = upper_bound(v.begin(), v.end(), 7);  // 9 가리킴
```

|함수|반환|예시 (target=7)|
|---|---|---|
|`lower_bound`|첫 번째 **≥** target 위치|7의 위치|
|`upper_bound`|첫 번째 **>** target 위치|9의 위치|

### 매개변수 탐색 (Parametric Search)

> **조건을 만족하는 최솟값/최댓값을 이진 탐색으로 찾기**

```cpp
// 예: 최소 X일 동안 목표 생산량 달성 가능한지?
bool check(int days) {
    long long total = 0;
    for (int rate : rates) total += (long long)days * rate;
    return total >= target;
}

int lo = 1, hi = 1e9, ans = hi;
while (lo <= hi) {
    int mid = lo + (hi - lo) / 2;
    if (check(mid)) { ans = mid; hi = mid - 1; }
    else lo = mid + 1;
}
```

### 코드포스 예시: Codeforces 1201C

```cpp
// 배열에서 조건 만족하는 최소 k 찾기
#include <bits/stdc++.h>
using namespace std;

int n;
vector<int> a;

bool check(int k) {
    int cnt = 0;
    for (int x : a) if (x >= k) cnt++;
    return cnt * k >= n;  // 조건 판단
}

int main() {
    cin >> n;
    a.resize(n);
    for (auto& x : a) cin >> x;
    sort(a.begin(), a.end());

    int lo = 1, hi = *max_element(a.begin(), a.end()), ans = 0;
    while (lo <= hi) {
        int mid = lo + (hi - lo) / 2;
        if (check(mid)) { ans = mid; lo = mid + 1; }
        else hi = mid - 1;
    }
    cout << ans;
}
```

## 2. 투 포인터 (Two Pointers)

> **두 개의 포인터를 사용해 탐색 범위를 줄이는 방법**

```
시간복잡도: O(N)
전제 조건:  보통 정렬된 배열 또는 연속 구간 문제
```

### 유형 1: 양 끝 포인터 (합 찾기)

```cpp
// 정렬된 배열에서 합이 target인 두 수 찾기
vector<int> v = {1, 2, 3, 4, 5, 6, 7};
int target = 9;

int left = 0, right = v.size() - 1;

while (left < right) {
    int sum = v[left] + v[right];

    if (sum == target) {
        cout << v[left] << " " << v[right];
        break;
    }
    else if (sum < target) left++;   // 합이 작으면 왼쪽 포인터 이동
    else right--;                    // 합이 크면 오른쪽 포인터 이동
}
```

### 유형 2: 같은 방향 포인터 (연속 구간)

```cpp
// 합이 target인 연속 부분배열 개수
vector<int> v = {1, 2, 3, 4, 5};
int target = 5, cnt = 0;

int left = 0, sum = 0;
for (int right = 0; right < v.size(); right++) {
    sum += v[right];

    while (sum > target) {
        sum -= v[left];
        left++;
    }
    if (sum == target) cnt++;
}
cout << cnt;
```

### 유형 3: 정렬 후 투 포인터

```cpp
// 두 배열에서 공통 원소 찾기
vector<int> a = {1, 3, 5, 7}, b = {2, 3, 6, 7};
sort(a.begin(), a.end());
sort(b.begin(), b.end());

int i = 0, j = 0;
while (i < a.size() && j < b.size()) {
    if (a[i] == b[j]) { cout << a[i] << " "; i++; j++; }
    else if (a[i] < b[j]) i++;
    else j++;
}
```

### 코드포스 예시: Codeforces 1462D

```cpp
// 합이 0 이상인 연속 부분배열 최대 길이
#include <bits/stdc++.h>
using namespace std;

int main() {
    int t; cin >> t;
    while (t--) {
        int n; cin >> n;
        vector<int> a(n);
        for (auto& x : a) cin >> x;

        int left = 0, ans = 0;
        long long sum = 0;

        for (int right = 0; right < n; right++) {
            sum += a[right];
            while (sum < 0) sum -= a[left++];
            ans = max(ans, right - left + 1);
        }
        cout << ans << "\n";
    }
}
```

## 3. 탐욕 알고리즘 (Greedy)

> **현재 상황에서 가장 좋아 보이는 선택을 반복하는 방법**

```
시간복잡도: 문제마다 다름 (보통 O(N log N) 이하)
핵심:       각 단계의 최선이 전체 최선이 되는 경우에만 성립
```

### 탐욕 알고리즘 성립 조건

```
탐욕적 선택 속성: 현재의 선택이 이후에 영향을 주지 않음
최적 부분 구조:  부분 문제의 최적해가 전체 문제의 최적해
```

### 대표 패턴 1: 정렬 후 탐욕

```cpp
// 회의실 배정: 최대한 많은 회의 배정
// 끝나는 시간 기준 오름차순 정렬
vector<pair<int,int>> meetings = {{1,4},{3,5},{0,6},{5,7},{3,8},{5,9},{6,10},{8,11},{8,12},{2,13},{12,14}};

sort(meetings.begin(), meetings.end(), [](auto& a, auto& b){
    return a.second < b.second;  // 끝 시간 기준 정렬
});

int cnt = 0, endTime = -1;
for (auto [s, e] : meetings) {
    if (s >= endTime) {  // 이전 회의가 끝난 후 시작
        cnt++;
        endTime = e;
    }
}
cout << cnt;
```

### 대표 패턴 2: 우선순위 큐 탐욕

```cpp
// 최소 비용으로 모든 작업 처리
#include <queue>

priority_queue<int, vector<int>, greater<int>> pq;  // 최소 힙
for (int cost : costs) pq.push(cost);

long long total = 0;
while (!pq.empty()) {
    total += pq.top();
    pq.pop();
}
```

### 대표 패턴 3: 거스름돈 문제

```cpp
// 동전 최소 개수 (큰 동전부터)
vector<int> coins = {500, 100, 50, 10};
int change = 1260, cnt = 0;

for (int coin : coins) {
    cnt += change / coin;
    change %= coin;
}
cout << cnt;  // 6
```

### 코드포스 예시: Codeforces 1374C

```cpp
// 배열 원소 최소 연산으로 정렬
#include <bits/stdc++.h>
using namespace std;

int main() {
    int t; cin >> t;
    while (t--) {
        int n; cin >> n;
        vector<int> a(n);
        for (auto& x : a) cin >> x;

        // 뒤에서부터 내림차순 유지되는 최장 구간 찾기
        int k = n - 1;
        while (k > 0 && a[k] >= a[k-1]) k--;
        // k 이전 원소들만 정렬하면 됨
        sort(a.begin(), a.begin() + k);

        // 정렬 후 전체가 비내림차순인지 확인
        bool ok = true;
        for (int i = 1; i < n; i++)
            if (a[i] < a[i-1]) { ok = false; break; }

        cout << (ok ? k : -1) << "\n";
    }
}
```

## 4. 슬라이딩 윈도우 (Sliding Window)

> **고정 크기 또는 가변 크기 구간을 이동시키며 탐색**

```
시간복잡도: O(N)
BFS의 deque 최적화에도 활용
```

### 고정 크기 윈도우

```cpp
// 크기 k인 연속 부분배열의 최대 합
vector<int> v = {1, 3, -1, -3, 5, 3, 6, 7};
int k = 3;

int window = 0, maxSum = INT_MIN;

// 첫 윈도우 초기화
for (int i = 0; i < k; i++) window += v[i];
maxSum = window;

// 윈도우 슬라이딩
for (int i = k; i < v.size(); i++) {
    window += v[i] - v[i - k];  // 오른쪽 추가, 왼쪽 제거
    maxSum = max(maxSum, window);
}
cout << maxSum;  // 16
```

### 가변 크기 윈도우

```cpp
// 합이 target 이상인 최소 길이 연속 부분배열
vector<int> v = {2, 3, 1, 2, 4, 3};
int target = 7, ans = INT_MAX;

int left = 0, sum = 0;
for (int right = 0; right < v.size(); right++) {
    sum += v[right];
    while (sum >= target) {
        ans = min(ans, right - left + 1);
        sum -= v[left++];
    }
}
cout << (ans == INT_MAX ? 0 : ans);  // 2
```

### deque를 이용한 슬라이딩 윈도우 최솟값

```cpp
// 크기 k인 모든 윈도우의 최솟값
#include <deque>
vector<int> v = {1, 3, -1, -3, 5, 3, 6, 7};
int k = 3;
deque<int> dq;  // 인덱스 저장

for (int i = 0; i < v.size(); i++) {
    // 범위 벗어난 인덱스 제거
    if (!dq.empty() && dq.front() <= i - k) dq.pop_front();

    // 현재값보다 큰 값 제거 (단조 덱)
    while (!dq.empty() && v[dq.back()] >= v[i]) dq.pop_back();

    dq.push_back(i);

    if (i >= k - 1) cout << v[dq.front()] << " ";
}
// -1 -3 -3 -3 3 3
```

### 코드포스 예시: Codeforces 1702D

```cpp
// 모든 윈도우에서 조건 만족하는 최소 k 찾기
#include <bits/stdc++.h>
using namespace std;

int main() {
    int t; cin >> t;
    while (t--) {
        int n, k; cin >> n >> k;
        string s; cin >> s;

        int zeros = 0, ans = 0;
        // 첫 윈도우
        for (int i = 0; i < k; i++) if (s[i] == '0') zeros++;
        ans = zeros;

        for (int i = k; i < n; i++) {
            if (s[i] == '0') zeros++;
            if (s[i - k] == '0') zeros--;
            ans = min(ans, zeros);
        }
        cout << ans << "\n";
    }
}
```

## 5. 삼분 탐색 (Ternary Search)

> **단봉 함수(볼록/오목)에서 최솟값/최댓값을 찾는 방법**

```
시간복잡도: O(log N)
전제 조건:  단봉 함수 (최솟값/최댓값이 하나)
```

```
      *
     / \
    /   \
   /     \
  /       \
← 탐색 범위 →
```

### 실수 삼분 탐색

```cpp
double ternarySearch(double lo, double hi) {
    for (int iter = 0; iter < 200; iter++) {
        double m1 = lo + (hi - lo) / 3;
        double m2 = hi - (hi - lo) / 3;

        if (f(m1) < f(m2)) hi = m2;  // 최솟값 탐색
        else lo = m1;
    }
    return (lo + hi) / 2;
}
```

### 정수 삼분 탐색

```cpp
int ternarySearch(int lo, int hi) {
    while (hi - lo >= 3) {
        int m1 = lo + (hi - lo) / 3;
        int m2 = hi - (hi - lo) / 3;

        if (f(m1) <= f(m2)) hi = m2;
        else lo = m1;
    }
    // 남은 구간에서 최솟값 탐색
    int ans = lo;
    for (int i = lo; i <= hi; i++)
        if (f(i) < f(ans)) ans = i;
    return ans;
}
```

### 코드포스 예시: Codeforces 439D

```cpp
// 단봉 함수에서 최솟값 위치 찾기
#include <bits/stdc++.h>
using namespace std;

int n;
vector<int> a;

long long f(long long x) {
    long long res = 0;
    for (int v : a) res += abs(v - x);
    return res;
}

int main() {
    cin >> n;
    a.resize(n);
    for (auto& x : a) cin >> x;

    long long lo = *min_element(a.begin(), a.end());
    long long hi = *max_element(a.begin(), a.end());

    while (hi - lo >= 3) {
        long long m1 = lo + (hi - lo) / 3;
        long long m2 = hi - (hi - lo) / 3;
        if (f(m1) <= f(m2)) hi = m2;
        else lo = m1;
    }

    long long ans = LLONG_MAX;
    for (long long x = lo; x <= hi; x++) ans = min(ans, f(x));
    cout << ans;
}
```

## 6. 분리 집합 / 유니온 파인드 (Union-Find)

> **원소들의 집합 관계를 빠르게 관리하는 자료구조**

```
시간복잡도: O(α(N)) ≈ O(1) (경로 압축 + 랭크)
주요 용도:  그래프 사이클 감지, 연결 요소, 최소 신장 트리
```

### 기본 구현

```cpp
const int MAX = 1001;
int parent[MAX];
int rank_[MAX];

// 초기화
void init(int n) {
    for (int i = 1; i <= n; i++) {
        parent[i] = i;
        rank_[i] = 0;
    }
}

// 루트 찾기 (경로 압축)
int find(int x) {
    if (parent[x] != x)
        parent[x] = find(parent[x]);  // 경로 압축
    return parent[x];
}

// 합치기 (랭크 기반)
void unite(int a, int b) {
    a = find(a); b = find(b);
    if (a == b) return;  // 이미 같은 집합

    if (rank_[a] < rank_[b]) swap(a, b);
    parent[b] = a;
    if (rank_[a] == rank_[b]) rank_[a]++;
}

// 같은 집합인지 확인
bool same(int a, int b) {
    return find(a) == find(b);
}
```

### 사이클 감지

```cpp
// 간선 추가 전 같은 집합이면 → 사이클
bool hasCycle = false;
for (auto [a, b] : edges) {
    if (same(a, b)) { hasCycle = true; break; }
    unite(a, b);
}
```

### 코드포스 예시: Codeforces 1374F

```cpp
// 연결 요소 수 관리
#include <bits/stdc++.h>
using namespace std;

int parent[100001];
int find(int x) { return parent[x] == x ? x : parent[x] = find(parent[x]); }

int main() {
    int n, q; cin >> n >> q;
    for (int i = 1; i <= n; i++) parent[i] = i;

    int components = n;
    while (q--) {
        int type, a, b; cin >> type >> a >> b;
        if (type == 1) {
            if (find(a) != find(b)) {
                parent[find(a)] = find(b);
                components--;
            }
        } else {
            cout << (find(a) == find(b) ? "YES" : "NO") << "\n";
        }
    }
}
```

## 알고리즘 선택 가이드

|상황|알고리즘|
|---|---|
|정렬된 배열에서 값 찾기|이진 탐색|
|조건 만족하는 최소/최댓값|매개변수 탐색|
|두 값의 합/연속 구간|투 포인터|
|고정/가변 구간 최적화|슬라이딩 윈도우|
|현재 최선 선택 반복|탐욕|
|단봉 함수 최솟값/최댓값|삼분 탐색|
|집합 관계 / 사이클 감지|유니온 파인드|

---

## 🔑 핵심 정리

```
이진 탐색:    정렬 필수, mid = lo+(hi-lo)/2, O(log N)
투 포인터:    두 포인터 이동, O(N), 연속 구간 / 두 수 합
탐욕:         현재 최선 반복, 성립 조건 확인 필수
슬라이딩:     구간 이동, O(N), deque로 최솟값 최적화
삼분 탐색:    단봉 함수 전제, O(log N)
유니온파인드: 경로압축+랭크, O(α(N))≈O(1), 사이클 감지
```
