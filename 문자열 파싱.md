## C++ 문자열 파싱

**파싱(Parsing)** = 문자열에서 원하는 데이터를 꺼내는 것

> 예: `"홍길동,20,서울"` → 이름, 나이, 도시를 각각 분리

---

## 주요 방법 4가지

### 1. `stringstream` — 공백/특수문자로 분리할 때

```cpp
#include <sstream>
#include <string>
#include <iostream>
using namespace std;

int main() {
    string line = "홍길동 20 서울";
    stringstream ss(line);

    string name, city;
    int age;

    ss >> name >> age >> city;  // 공백 기준으로 자동 분리

    cout << name << endl;  // 홍길동
    cout << age  << endl;  // 20
    cout << city << endl;  // 서울
}
```

> 💡 `>>` 연산자가 공백을 기준으로 알아서 잘라줌

---

### 2. `getline` — 구분자로 분리할 때 (CSV 등)

```cpp
#include <sstream>
#include <string>
#include <iostream>
using namespace std;

int main() {
    string line = "홍길동,20,서울";
    stringstream ss(line);
    string token;

    while (getline(ss, token, ',')) {  // ',' 기준으로 자름
        cout << token << endl;
    }
    // 출력:
    // 홍길동
    // 20
    // 서울
}
```

---

### 3. `find` + `substr` — 직접 위치 찾아서 자르기

```cpp
#include <string>
#include <iostream>
using namespace std;

int main() {
    string s = "Hello:World";

    int pos = s.find(':');         // ':' 위치 찾기 → 5
    string left  = s.substr(0, pos);      // 0부터 5개  → "Hello"
    string right = s.substr(pos + 1);     // 6부터 끝까지 → "World"

    cout << left  << endl;  // Hello
    cout << right << endl;  // World
}
```

> 💡 `substr(시작위치, 길이)` — 길이 생략하면 끝까지

---

### 4. 숫자 변환 — `stoi`, `stod`, `to_string

```cpp
#include <string>
#include <iostream>
using namespace std;

int main() {
    string s = "42";

    int    n = stoi(s);      // string → int
    double d = stod(s);      // string → double
    string t = to_string(n); // int    → string

    cout << n + 1 << endl;   // 43
}
```

|함수|변환 방향|예시|
|---|---|---|
|`stoi()`|string → int|`stoi("42")` → 42|
|`stod()`|string → double|`stod("3.14")` → 3.14|
|`stol()`|string → long|`stol("100000")`|
|`to_string()`|숫자 → string|`to_string(42)` → "42"|

## 실전 예제 — CSV 한 줄 파싱

cpp

```cpp
#include <sstream>
#include <string>
#include <vector>
#include <iostream>
using namespace std;

int main() {
    string line = "Alice,30,Engineer";
    stringstream ss(line);
    string token;
    vector<string> result;

    while (getline(ss, token, ',')) {
        result.push_back(token);
    }

    string name = result[0];       // "Alice"
    int    age  = stoi(result[1]); // 30
    string job  = result[2];       // "Engineer"

    cout << name << " / " << age << " / " << job << endl;
}
```

## 방법 선택 가이드

```
공백으로 구분된 데이터    → stringstream + >>
쉼표/특수문자로 구분      → getline(ss, token, 구분자)
특정 패턴 위치 찾기       → find + substr
문자열 ↔ 숫자 변환        → stoi / stod / to_string
```
