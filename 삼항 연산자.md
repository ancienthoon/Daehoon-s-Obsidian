# ⚖️ C++ 삼항 연산자 (조건 연산자)

> [!abstract] **한 줄 요약** `조건 ? 참일_때_값 : 거짓일_때_값` "조건이 맞니? 맞으면 앞의 것을, 아니면 뒤의 것을 선택해!"라고 명령하는 연산자입니다.

---

## 1. 문법 구조 (Syntax)

C++에서 유일하게 **세 개의 항**을 사용하는 연산자라서 '삼항' 연산자라고 부릅니다.

```
자료형 변수명 = (조건식) ? (참일 때의 값) : (거짓일 때의 값);
```

- **`?` 앞:** 참인지 거짓인지 판별할 **조건**이 옵니다.
    
- **`?` 와 `:` 사이:** 조건이 **참(true)**일 때 반환할 값입니다.
    
- **`:` 뒤:** 조건이 **거짓(false)**일 때 반환할 값입니다.
    

---

## 2. 왜 사용하는가? (if-else와 비교)

동일한 기능을 하는 코드를 비교해 보면 삼항 연산자의 간결함을 알 수 있습니다.

### ❌ 일반적인 if-else 방식

```cpp
int a = 10, b = 20;
int max_val;

if (a > b) {
    max_val = a;
} else {
    max_val = b;
}
```

### ✅ 삼항 연산자 방식

```cpp
int a = 10, b = 20;
int max_val = (a > b) ? a : b;
```

---

## 3. 실전 활용 및 주의사항

### ① 출력문 내에서 바로 사용하기

백준 문제 풀이 시, 조건에 따라 "YES"나 "NO"를 출력해야 할 때 매우 유용합니다.

```cpp
cout << (is_success ? "YES" : "NO") << "\n";
```

### ② 중첩 사용 (권장하지 않음)

삼항 연산자 안에 또 삼항 연산자를 넣을 수 있습니다. 하지만 코드가 너무 복잡해져서 읽기 힘들어지므로 가급적 피하는 것이 좋습니다.

```cpp
// a가 양수인지, 음수인지, 0인지 판별 (가독성이 떨어짐)
string res = (a > 0) ? "positive" : (a < 0) ? "negative" : "zero";
```

## 💻 실전 코드 예시

사용자님의 스타일(입출력 최적화, 소문자 변수명, `using namespace std`)을 적용한 예제입니다.

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    ios_base::sync_with_stdio(false); cin.tie(NULL); cout.tie(NULL);

    int n;
    cin >> n;

    // 짝수인지 홀수인지 판별하여 바로 대입
    string result = (n % 2 == 0) ? "even" : "odd";

    cout << n << " is " << result << "\n";

    // 두 수 중 더 큰 값 찾기
    int x = 50, y = 100;
    int big = (x > y) ? x : y;
    
    cout << "larger value: " << big << "\n";

    return 0;
}
```

## 🎓 논리적 설명 (중학생 수준)

삼항 연산자는 **"갈림길에서 표지판을 보는 것"**과 같습니다.

1. 갈림길 앞에 **조건**이라는 표지판이 있습니다. (`n % 2 == 0`)
    
2. 표지판 내용이 맞으면 **왼쪽 길(`?` 쪽)**로 가서 선물을 챙깁니다.
    
3. 표지판 내용이 틀리면 **오른쪽 길(`:` 쪽)**로 가서 선물을 챙깁니다.
    
4. 결국 우리는 한 줄의 코드로 원하는 선물을 바로 변수에 담을 수 있게 됩니다.
